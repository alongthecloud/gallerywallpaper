# Gallery live-wallpaper (갤러리 라이브-배경화면)

### 앱에 대하여
사용된 언어 및 플랫폼
* Flutter(Dart)
* Android(Java)

버전
ver 1.0.0

### 만든 목적
이 앱은 라이브 배경화면의 서비스 구현 부분은 Android native java 로 만들고 설정 UI 부분을 Flutter 로 작성한 앱입니다.

이 앱은
* Flutter 를 사용하여 UI 를 효과적으로 만듦.
* Flutter 를 사용하여 native 와 연결되어 동작.

두 가지 목적을 가지고 만들었습니다.
